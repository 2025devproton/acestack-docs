# AceStack Docs

This repo contains the source code for the AceStack documentation.

To contribute to the documentation:

1. Clone the repo
   ```git clone https://github.com/acestack-org/acestack-docs.git```
2. Install Zensical (with Pip)
    ```
    python3 -m venv .venv
    source .venv/bin/activate
    pip install zensical
    ```
3. Deploy the live testing environment
    ```
    zensical serve
    ```
